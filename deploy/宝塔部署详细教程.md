# 织里镇童装产业协同管理平台 — 宝塔面板部署详细教程

---

## 前置准备

在开始之前，你需要有一台安装了宝塔面板的云服务器。

### 服务器配置建议

| 项目 | 最低配置 | 推荐配置 |
|---|---|---|
| CPU | 1核 | 2核+ |
| 内存 | 2GB | 4GB+ |
| 硬盘 | 40GB | 60GB+ |
| 系统 | CentOS 7.9 | CentOS 7.9 / Ubuntu 20.04 |

### 你需要准备的东西

1. **服务器 IP 地址** — 云服务器公网 IP
2. **宝塔面板地址** — 通常是 `http://你的IP:8888`
3. **宝塔面板账号密码** — 安装宝塔时生成的那个
4. **域名**（可选）— 有域名更好，没有就用 IP 访问
5. **QQ邮箱 + SMTP授权码** — 用于发送注册验证码、找回密码邮件
6. **DeepSeek API Key** — AI助手功能需要，去 [platform.deepseek.com](https://platform.deepseek.com) 注册获取
7. **阿里云 OSS AccessKey** — 用于头像/营业执照图片上传（如果没有，先去阿里云开通 OSS）

## 第0步：确认部署包文件

在你自己电脑上，确认 `deploy/` 目录下有以下文件：

```
deploy/
├── zhili-kids-industry.jar          ← Spring Boot 后端（46MB）
├── application.yaml                 ← 后端配置文件
├── nginx-bt.conf                    ← Nginx 配置文件
├── start.sh                         ← 启动脚本
├── stop.sh                          ← 停止脚本
├── sql/
│   ├── init.sql                     ← 数据库表结构
│   └── inventory_tables.sql         ← 库存模块表
├── 部署说明.md
└── zhili-kids-industry-system/      ← 前端页面（几十个文件）
```

**缺少 deploy 目录？** 在你电脑的项目目录下运行：
```bash
cd ~/Desktop/Project/java
mvn clean package -DskipTests
```
打包完成后 `deploy/` 目录就会生成。

## 第1步：登录宝塔面板

1. 浏览器打开 `http://你的服务器IP:8888`
2. 输入宝塔面板的账号和密码
3. 进入面板首页

**如果你还没有安装宝塔面板**，SSH 连上服务器后执行：

CentOS 安装命令：
```bash
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
```

Ubuntu/Debian 安装命令：
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```

安装完成后终端会显示面板地址、账号、密码，保存好。

## 第2步：安装基础软件

进入宝塔面板首页，会弹出"推荐安装"的窗口。如果没有，点击左侧菜单**「软件商店」**。

### 2.1 安装 Nginx

1. 软件商店搜索 **Nginx**
2. 点击 **安装**，选择版本 **1.24** 或最新稳定版
3. 选择**极速安装**（不用编译安装）
4. 等待安装完成，约 1-2 分钟

### 2.2 安装 MySQL

1. 软件商店搜索 **MySQL**
2. 选择 **MySQL 8.0**（推荐）或 MySQL 5.7
3. 点击 **安装** → **极速安装**
4. 等待安装完成，约 3-5 分钟

**安装时会让设置 root 密码，请务必记下来！**

### 2.3 安装 Redis

1. 软件商店搜索 **Redis**
2. 选择 **Redis 7.0** 或最新版
3. 点击 **安装**
4. 等待安装完成，约 1 分钟

安装完成后 Redis 默认没有密码，可以不改。如果后续想设密码：
- 点击 Redis → 设置 → 性能调整 → requirepass 设置一个密码
- 记住这个密码，后面要写到配置文件里

### 2.4 安装 Java（JDK 17）

1. 软件商店搜索 **Java**
2. 找到 **Java项目一键部署**（这个插件会自动装JDK）
3. 点击 **安装**

或者手动安装 JDK：
- 软件商店搜索 **JDK**
- 选择 **JDK17** 安装

## 第3步：上传项目文件

### 3.1 创建项目目录

1. 宝塔面板左侧点击**「文件」**
2. 进入 `/www/wwwroot/` 目录
3. 点击上方 **新建** → **新建目录**
4. 目录名填：`zhili-kids-industry`
5. 点击确定

### 3.2 上传文件

进入 `/www/wwwroot/zhili-kids-industry/` 目录，上传以下文件：

**方式一：宝塔文件管理器上传（推荐小文件）**

1. 点击**上传**按钮
2. 选择你本机 `deploy/` 下的文件
3. 一次最多传几个文件，分多次传

**方式二：整个目录打包上传（推荐）**

1. 在你本机，把 `deploy/` 里的内容打成一个 zip 包
2. 宝塔文件管理器进入 `/www/wwwroot/zhili-kids-industry/`
3. 上传这个 zip 包
4. 上传完后，宝塔里右键这个 zip → **解压** → 解压到当前目录
5. 删除 zip 包

### 3.3 验证上传结果

上传完后，目录结构应该是这样：

```
/www/wwwroot/zhili-kids-industry/
├── zhili-kids-industry.jar
├── application.yaml
├── nginx-bt.conf
├── start.sh
├── stop.sh
├── sql/
│   ├── init.sql
│   └── inventory_tables.sql
└── zhili-kids-industry-system/
    ├── index.html
    ├── 50x.html
    ├── pages/
    │   ├── login.html
    │   ├── register.html
    │   └── ... (很多html)
    ├── js/
    │   └── ... (很多js)
    ├── css/
    │   └── ... (很多css)
    └── images/
        └── ...
```

## 第4步：创建数据库并导入表结构

### 4.1 在宝塔面板创建数据库

1. 宝塔左侧菜单 → **「数据库」**
2. 点击 **添加数据库**
3. 填写以下信息：

| 字段 | 填写内容 |
|---|---|
| 数据库名 | `zhili_kids_industry` |
| 用户名 | `zhili_kids_industry` |
| 密码 | 自己设置一个复杂点的密码**（记下来！）** |
| 访问权限 | 本地服务器 |
| 字符集 | `utf8mb4` |

4. 点击 **提交**

### 4.2 导入数据库表结构

1. 在数据库列表里，找到刚创建的 `zhili_kids_industry`
2. 点击右侧的 **管理** 按钮（会打开 phpMyAdmin）
3. 在 phpMyAdmin 左侧选中 `zhili_kids_industry` 数据库
4. 点击顶部菜单 **SQL**
5. 打开你自己电脑上 `deploy/sql/init.sql`，全选复制内容
6. 粘贴到 phpMyAdmin 的 SQL 输入框
7. 点击右下角 **执行**
8. 看到绿色成功提示，说明导入成功
9. 同样的方式，复制 `deploy/sql/inventory_tables.sql` 内容到 SQL 框，执行

执行完 `init.sql` 后，会自动：
- 创建所有需要的表（用户表、商品表、订单表等）
- 插入一个管理员账号（邮箱：`admin@admin.com`，密码：`admin123`）

**重要：登录后记得修改管理员密码！**

## 第5步：修改应用配置

这是最关键的一步。在宝塔文件管理器中，找到并编辑文件：

**`/www/wwwroot/zhili-kids-industry/application.yaml`**

### 5.1 数据库配置（必改）

找到这一段：
```yaml
spring:
  datasource:
    username: zhili_kids_industry
    password: YOUR_DB_PASSWORD
```

把 `YOUR_DB_PASSWORD` 改成第4步创建数据库时设置的密码：
```yaml
    username: zhili_kids_industry
    password: 你的真实密码
```

### 5.2 邮件配置（必改 — 不配注册功能不可用）

登录 QQ邮箱 → 设置 → 账户 → POP3/SMTP服务 → 开启 → 生成授权码。

```yaml
spring.mail:
  host: smtp.qq.com
  port: 587
  username: 你的QQ号@qq.com      # 改这里
  password: 你的QQ邮箱SMTP授权码   # 改这里（不是QQ密码！）
```

> 如果你用其他邮箱（163/Gmail等），自行搜索对应的 SMTP 服务器地址和端口。

### 5.3 Redis 配置（通常不用改）

```yaml
spring.data.redis:
  host: 127.0.0.1
  port: 6379
  password:          # 如果你在第2步给Redis设了密码，填这里；否则留空
  database: 0
```

### 5.4 DeepSeek API Key（AI助手，可选）

去 [platform.deepseek.com](https://platform.deepseek.com) → 注册/登录 → API Keys → 创建新Key → 复制。

```yaml
deepseek:
  api-key: sk-xxxxxxxxxxxxxxxx    # 改成你的
```

### 5.5 阿里云 DashScope API Key（图片识别，可选）

```yaml
dashscope:
  api-key: sk-xxxxxxxxxxxxxxxx    # 改成你的
```

### 5.6 阿里云 OSS（文件上传，可选）

需要在阿里云创建 Bucket 和 RAM 账号，生成 AccessKey：

```yaml
aliyun:
  oss:
    access-key-id: LTAI5xxxxxxxxx
    access-key-secret: xxxxxxxxxxxxxxxx
```

> 阿里云 OSS 配置比较麻烦。如果暂时不用文件上传功能（头像、营业执照），可以先保持默认值，不影响核心功能。

### 5.7 修改完成后保存

宝塔编辑器点击**保存**按钮。

## 第6步：配置 Nginx

### 6.1 创建网站

1. 宝塔左侧菜单 → **「网站」**
2. 点击 **添加站点**
3. 填写信息：

| 字段 | 值 |
|---|---|
| 域名 | 填你的域名，没有就填服务器IP（如 `1.2.3.4`） |
| 根目录 | `/www/wwwroot/zhili-kids-industry/zhili-kids-industry-system` |
| FTP | 不创建 |
| 数据库 | 不创建 |
| PHP版本 | 纯静态 |

4. 点击 **提交**

### 6.2 替换 Nginx 配置

1. 网站列表里找到刚创建的站点
2. 点击右侧 **设置** → **配置文件**
3. 你会看到宝塔生成的默认配置
4. 全选删除
5. 打开你本机 `deploy/nginx-bt.conf`，全选复制
6. 粘贴到宝塔的配置文件编辑器
7. 找到这一行：
   ```
   server_name  your-domain.com;
   ```
   改成你的实际域名或IP，如：
   ```
   server_name  1.2.3.4;
   ```
8. 点击 **保存**

### 6.3 重载 Nginx

保存后宝塔会自动检测配置是否正确。如果提示配置有误，检查域名/IP是否写对了。

配置正确后，Nginx 会自动重载。如果没有，手动点一下 **重载配置**。

## 第7步：启动后端服务

### 方式一：用脚本启动（推荐）

1. 宝塔左侧菜单 → **「终端」**（或者用你本机的 SSH 工具连上服务器）
2. 执行以下命令：

```bash
cd /www/wwwroot/zhili-kids-industry
chmod +x start.sh stop.sh
./start.sh
```

3. 看到 `启动成功!` 说明后端启动了

4. 检查是否真的跑起来了：
```bash
curl http://localhost:8080/api/health
```
如果返回 JSON 数据，说明正常。

### 方式二：用宝塔 Java 项目一键部署

1. 宝塔左侧 → **「Java项目一键部署」**（如果没有这个菜单，去软件商店安装插件）
2. 点击 **添加Java项目**
3. 填写：

| 字段 | 值 |
|---|---|
| 项目JAR路径 | `/www/wwwroot/zhili-kids-industry/zhili-kids-industry.jar` |
| 项目端口 | `8080` |
| JDK版本 | `JDK17` |
| 启动参数 | `--spring.config.location=/www/wwwroot/zhili-kids-industry/application.yaml` |
| 项目名称 | `zhili-kids-industry` |

4. 点击 **提交** 然后点 **启动**

### 查看日志

如果启动失败，查看日志定位问题：

```bash
# 方式一的日志
cat /www/wwwroot/zhili-kids-industry/logs/app.log

# 方式二的日志
# 在 Java项目一键部署 页面，点击日志按钮
```

## 第8步：放行端口

1. 宝塔左侧 → **「安全」**
2. 检查以下端口状态：

| 端口 | 说明 | 操作 |
|---|---|---|
| 80 | Nginx / 前端访问 | 默认已放行，如没有就添加 |
| 443 | HTTPS | 如果需要HTTPS就放行 |
| 8080 | 后端API | 可以不放行（通过Nginx访问即可） |
| 8888 | 宝塔面板 | 默认已放行 |

3. 如果用的云服务器（阿里云/腾讯云等），还需要去**云服务商的安全组**里放行 80 端口：
   - 阿里云：ECS 控制台 → 安全组 → 添加入方向规则 → 端口 80/0 → 授权对象 0.0.0.0/0
   - 腾讯云：CVM 控制台 → 安全组 → 添加规则 → 同上

## 第9步：验证部署

### 9.1 浏览器访问

打开浏览器，输入：
```
http://你的服务器IP
```

如果你配了域名，访问：
```
http://你的域名
```

### 9.2 检查清单

| 检查项 | 预期结果 |
|---|---|
| 打开首页 | 能看到织里镇童装平台首页，有轮播图、导航栏 |
| 点击"登录" | 跳转到登录页面 |
| 注册新账号 | 能收到邮箱验证码（前提是SMTP配好了） |
| 点击"产品展示" | 能看到产品列表 |
| 点击"市场信息" | 能看到市场数据页面 |
| 点击"数据中心" | 数据正常加载 |
| 打开F12控制台 | 没有红色的报错（黄色警告可以忽略） |

### 9.3 如果没有显示或报错

常见原因：

1. **页面白屏/404**：
   - 检查 Nginx 是否启动：宝塔首页 → 看 Nginx 状态灯是绿色
   - 检查根目录是否选择正确

2. **页面打开但数据加载不出来**：
   - 检查后端是否启动：`ps -ef | grep zhili`
   - 检查后端日志：`cat /www/wwwroot/zhili-kids-industry/logs/app.log`
   - 常见原因：数据库密码不对、MySQL没启动

3. **API 报 502**：
   - 说明后端没启动或 8080 端口没开
   - SSH 进去执行：`curl http://localhost:8080/api/health`

## 第10步：后续维护

### 更新部署

以后代码有更新时：

```bash
# 1. 停止旧服务
cd /www/wwwroot/zhili-kids-industry
./stop.sh

# 2. 上传新的 jar 包（替换旧的）

# 3. 如果数据库有变化，执行新的 SQL

# 4. 重新启动
./start.sh
```

### 设置开机自启

SSH 执行：
```bash
# 编辑 rc.local
vi /etc/rc.d/rc.local

# 添加这一行到最后：
cd /www/wwwroot/zhili-kids-industry && ./start.sh

# 给执行权限
chmod +x /etc/rc.d/rc.local
```

### 配置 HTTPS（SSL证书）

1. 宝塔 → 网站 → 你的站点 → **设置** → **SSL**
2. 选择 **Let's Encrypt** 或 **宝塔SSL**
3. 申请证书
4. 开启 **强制HTTPS**

### 每日备份数据库

1. 宝塔 → **计划任务** → 添加
2. 任务类型：**备份数据库**
3. 选择 `zhili_kids_industry`
4. 周期：每天凌晨 3 点
5. 保留份数：7 份

## 常见问题排查

### Q1：启动报 "Failed to configure a DataSource"

数据库连接不上。检查：
- MySQL 是否在运行（宝塔首页看 MySQL 状态）
- `application.yaml` 里的数据库密码是否正确
- 数据库 `zhili_kids_industry` 是否创建成功

### Q2：启动报 "Unable to connect to Redis"

Redis 没连上。检查：
- Redis 是否在运行
- `application.yaml` 里的 Redis 密码是否正确
- Redis 端口 6379 是否被占用

### Q3：注册时收不到验证码邮件

检查：
- QQ邮箱 SMTP 是否开启
- `application.yaml` 里的 `spring.mail.password` 是**授权码**不是QQ密码
- 邮件可能进了垃圾箱

### Q4：头像/营业执照上传失败

- OSS 的 AccessKey 不对或已失效
- OSS Bucket 不存在
- 跨域配置问题

### Q5：WebSocket 聊天收不到消息

WebSocket 需要 Nginx 特殊配置。检查 `nginx-bt.conf` 里是否有这一段：
```nginx
location /ws {
    proxy_pass http://127.0.0.1:8080;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "upgrade";
    ...
}
```

### Q6：80 端口被占用

宝塔 → 安全 → 看 80 端口是否被其他服务占用。如果是，停止那个服务或者换个端口。
