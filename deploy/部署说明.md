# 织里镇童装产业协同管理平台 — 宝塔面板部署指南

## 服务器要求

- CentOS 7+ / Ubuntu 18+ （推荐 CentOS 7.9）
- Java 17+
- MySQL 8.0+
- Redis 6.0+
- Nginx 1.20+

## 一、宝塔面板安装基础环境

1. 登录宝塔面板
2. 软件商店安装：
   - **Java项目一键部署** 插件（或使用启动脚本手动管理）
   - **MySQL 8.0**
   - **Redis 6.0+**
   - **Nginx 1.20+**（如果还没有）

## 二、上传文件

将整个 `deploy` 目录上传到服务器的 `/www/wwwroot/zhili-kids-industry/`

最终目录结构：
```
/www/wwwroot/zhili-kids-industry/
├── zhili-kids-industry.jar          # Spring Boot JAR
├── application.yaml                 # 生产环境配置（需要修改）
├── start.sh                         # 启动脚本
├── stop.sh                          # 停止脚本
├── sql/
│   ├── init.sql                     # 数据库初始化
│   └── inventory_tables.sql         # 库存相关表
└── zhili-kids-industry-system/      # 前端静态文件
    ├── index.html
    ├── pages/
    ├── js/
    ├── css/
    └── images/
```

## 三、创建数据库

1. 宝塔面板 → 数据库 → 添加数据库
   - 数据库名：`zhili_kids_industry`
   - 用户名：`zhili_kids_industry`
   - 密码：自行设置（记录好，后面要用）
   - 字符集：utf8mb4

2. 导入表结构：
   - 宝塔面板 → 数据库 → 管理 → 选择 `zhili_kids_industry` 库
   - 点击 SQL → 先上传 `sql/init.sql`，执行
   - 再上传 `sql/inventory_tables.sql`，执行

## 四、修改配置文件

编辑 `application.yaml`，把 `YOUR_xxx` 的占位值改成真实配置：

```yaml
# 数据库配置
spring.datasource:
  username: zhili_kids_industry
  password: 你刚才创建的数据库密码

# 邮件配置（用于发送验证码）
spring.mail:
  username: 你的QQ邮箱@qq.com
  password: QQ邮箱SMTP授权码

# 阿里云OSS（用于头像和营业执照上传）
aliyun.oss:
  access-key-id: 你的阿里云AK
  access-key-secret: 你的阿里云SK

# Redis（如果宝塔装的Redis有小密码）
spring.data.redis:
  password: Redis密码  # 没有密码就留空

# DeepSeek AI（AI助手功能）
deepseek:
  api-key: 你的DeepSeek API Key

# 阿里云DashScope（AI图片识别）
dashscope:
  api-key: 你的DashScope API Key
```

## 五、配置Nginx

1. 宝塔面板 → 网站 → 添加站点
   - 域名：填写你的域名或服务器IP
   - 根目录：选择 `/www/wwwroot/zhili-kids-industry/zhili-kids-industry-system`

2. 站点创建后 → 配置 → 配置文件，**用 nginx-bt.conf 的内容替换整个 server 块**

3. 把 `server_name` 改成你的实际域名

## 六、启动后端

```bash
cd /www/wwwroot/zhili-kids-industry
chmod +x start.sh stop.sh
./start.sh
```

启动后检查服务是否正常：
```bash
ps -ef | grep zhili-kids-industry
curl http://localhost:8080/api/health
```

## 七、防火墙放行端口

宝塔面板 → 安全：
- 放行 `8080`（后端API，可选，仅前端通过Nginx访问80即可）
- 放行 `80`（前端+Nginx）
- 放行 `443`（如果配置了HTTPS）

## 八、验证

1. 浏览器访问 `http://你的域名或IP`
2. 能看到首页即可
3. 注册一个账号，测试各功能

## 九、宝塔Java项目一键部署（备选方案）

如果不想用 start.sh 脚本，可以在宝塔的「Java项目一键部署」中添加：
- 项目JAR路径：`/www/wwwroot/zhili-kids-industry/zhili-kids-industry.jar`
- 项目端口：`8080`
- JDK版本：`JDK17`
- 项目名称：`zhili-kids-industry`

## 常见问题

1. **启动报 Redis 连接错误**：检查 Redis 是否启动，密码是否正确
2. **启动报 MySQL 连接错误**：检查数据库名、用户名、密码，确认MySQL运行中
3. **页面打不开**：检查宝塔防火墙是否放行80端口，Nginx是否启动
4. **图片上传失败**：检查阿里云OSS配置是否正确
5. **邮件验证码收不到**：检查QQ邮箱SMTP授权码是否正确
